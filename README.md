An interactive marimo notebook to explore how LLMs function as advanced autocomplete engines.

## How to Run
1. **Install dependencies**:
   This project uses `uv` for easy dependency management.
   ```bash
   uv sync
   ```
2. **Run the notebook**:
   Execute the following command to start the interactive marimo server:
   ```bash
   uv run marimo edit next_word_predictor.py
   ```
3. **Open in Browser**:
   Marimo will automatically open a tab in your default web browser (usually at `http://localhost:2718`).
